# جسر الويب للمتصفحات القديمة

جسر ويب خفيف يجلب المواقع الحديثة ويحوّلها إلى HTML كلاسيكي، مما يجعلها
قابلة للعرض على متصفحات قديمة مثل **Internet Explorer 2** (1995) حتى IE5،
و**Netscape Navigator 3/4**، على أنظمة **Windows 3.1** و95 و98.

يستخدم الجسر HTML 3.2 كمعيار أساسي، لكنه يذهب أبعد من ذلك لضمان التوافق
مع **المتصفحات الأقدم من HTML 3.2** مثل IE2:

- يتم **تغيير حجم الصور مسبقاً على مستوى الخادم** حتى تُعرض بشكل صحيح على
  المتصفحات التي تتجاهل خصائص `width`/`height` في HTML (أُضيفت في HTML 3.2)
- لا يُستخدم `data:` URIs (غير مدعوم قبل IE8) -- تُحوَّل صور SVG المضمّنة
  إلى JPEG وتُقدَّم عبر نقطة `/svg/` مخصصة
- صور JPEG فقط -- بدون PNG (غير مدعوم في IE3 وما قبله)
- بدون وسوم `<div>` أو `<span>` أو `<button>` (غير مدعومة في IE2)
- مسارات URL كاملة بدلاً من الترميز بعلامة النسبة المئوية (IE2 يُفسد `%XX`)

يزيل الجسر JavaScript وCSS والفيديو وSVG وعناصر HTML5، ثم يعيد بناء هيكل
الصفحة باستخدام وسوم `<table>` وخصائص HTML الكلاسيكية (`bgcolor`، `align`،
`<font>`، إلخ).

## المزايا

- **توافق من IE2 إلى IE5** -- يعمل مع متصفحات ما قبل HTML 3.2 (IE2) وما بعدها
- **مخرجات HTML كلاسيكية** -- بدون CSS أو JavaScript، تخطيط بالجداول
- **تصفح مواقع HTTPS** -- يتيح للحواسيب والمتصفحات القديمة التي لا تدعم التشفير الحديث (TLS 1.2/1.3) الوصول إلى مواقع HTTPS عبر الجسر، مع التراجع التلقائي إلى HTTP عند فشل HTTPS
- **تحويل تخطيطات CSS الحديثة** -- يحوّل grid و flex إلى جداول `<table>`
- **معالجة الصور** -- يجلب الصور ويحوّلها إلى JPEG ويعيد تحجيمها مسبقاً حسب الأبعاد المحددة في HTML وCSS (لتوافق IE2 الذي يتجاهل خصائص width/height)
- **تحويل SVG** -- يحوّل صور SVG المضمّنة والخارجية إلى JPEG عبر cairosvg، مع استخراج الأبعاد من viewBox وقواعد CSS
- **استخراج أبعاد الصور من CSS** -- يقرأ width و height و max-width من قواعد الأنماط ويطبّقها على الصور
- **دعم YouTube** -- يستخرج بيانات الفيديو والبحث من JSON المضمّن (YouTube معتمد بالكامل على JavaScript)
- **دعم صحيفة سبق** -- يستخرج مقالات الصفحة الرئيسية (أبرز الأخبار، اختيارات المحرر، الأكثر تداولاً) وصفحات المقالات من JSON المضمّن (sabq.org تطبيق JavaScript بالكامل)
- **دعم اللغة العربية والاتجاه من اليمين لليسار** -- يكتشف الصفحات العربية ويضبط `dir="rtl"`
- **ترميز CP-1256 تلقائي** -- يكتشف المحتوى العربي تلقائياً ويحوّله إلى Windows-1256 على الأنظمة القديمة، حتى لو لم يكن عنوان الموقع عربياً
- **إصلاح Windows 3.x** -- يفرض ترميز iso-8859-1 لحل مشكلة الصفحات الفارغة في IE5 على Windows 3.11
- **إزالة العناصر المخفية** -- يحذف العناصر ذات display:none (أزرار مكررة، قوائم إكمال تلقائي)
- **تحويل العروض الدوّارة** -- يحوّل owl-carousel و slick و swiper إلى جداول أفقية
- **معالجة النصوص غير القابلة للعرض** -- يزيل الأحرف غير اللاتينية وغير العربية التي تسبب مشاكل في العرض
- **لقطة شاشة** -- زر لالتقاط صورة للموقع الأصلي وعرضها كصورة JPEG
- **سجل التصفح** -- يحفظ العناوين المكتوبة يدوياً لكل مستخدم (حسب عنوان IP)
- **القوائم المنسدلة** -- يحوّل القوائم الحديثة إلى عناصر `<select>`
- **بديل بحث Google** -- يحوّل بحث Google إلى DuckDuckGo (لأن Google يتطلب JavaScript)
- **إعادة كتابة النماذج** -- يعيد توجيه النماذج للعمل من خلال الجسر
- **عزل الأقسام** -- يغلّف كل قسم في جدول مستقل لمنع تجاوز التخطيط
- **كشف CAPTCHA و SPA** -- يعيد المحاولة بهوية Googlebot للصفحات المحمية، ويكتشف صفحات JavaScript فقط
- **دعم Frameset** -- يكتشف صفحات `<frameset>` ويعيد كتابة عناوين الإطارات عبر الجسر
- **توافق عميق مع IE2** -- يزيل `<div>` و `<span>` و `<button>` و `<input type="file">`، يستخدم مسارات كاملة بدلاً من `%XX`، يحذف القوائم الفارغة وعناصر النماذج التي تعمل بالـ JavaScript فقط

## المتطلبات

- Python 3.6 أو أحدث
- حزم Python التالية:

| الحزمة              | الغرض                                | مطلوبة؟   |
|---------------------|--------------------------------------|-----------|
| `requests`          | جلب صفحات الويب                      | نعم       |
| `beautifulsoup4`    | تحليل وتحويل HTML                   | نعم       |
| `Pillow`            | تحويل الصور وتغيير حجمها            | اختياري   |
| `cairosvg`          | تحويل SVG إلى PNG/JPEG              | اختياري   |
| `selenium`          | لقطات الشاشة وعرض JavaScript        | اختياري   |
| `webdriver-manager` | تثبيت ChromeDriver تلقائياً          | اختياري   |

لميزة لقطات الشاشة، يجب تثبيت **Chromium** على الخادم:

```bash
# Debian / Ubuntu
sudo apt-get install -y chromium

# Fedora / RHEL
sudo dnf install -y chromium
```

## التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/techana/web-bridge-old-browsers.git
cd web-bridge-old-browsers

# تثبيت الاعتماديات الأساسية
pip install requests beautifulsoup4 Pillow cairosvg

# اختياري: تثبيت اعتماديات لقطات الشاشة وعرض JavaScript
pip install selenium webdriver-manager
```

## تشغيل الجسر

```bash
python3 web_bridge.py
```

يبدأ الجسر بالاستماع على **المنفذ 8888** على جميع واجهات الشبكة.

للتشغيل في الخلفية:

```bash
nohup python3 web_bridge.py &
```

## الاستخدام

### من المتصفح القديم

1. افتح متصفحك القديم (IE2، IE5، Netscape، إلخ)
2. انتقل إلى `http://<عنوان-الجسر>:8888` -- مثلاً `http://192.168.1.12:8888`
3. اكتب عنوان URL في حقل **العنوان** وانقر **Web Bridge**
4. يجلب الجسر الصفحة ويزيل الميزات الحديثة ويعيد نسخة HTML 3.2 مبسطة

### لقطات الشاشة

اختر دقة من قائمة **Screenshot** في شريط التنقل وانقر **>** لالتقاط صورة
للصفحة الأصلية بصيغة JPEG. تتراوح الدقات المتاحة من 640x480 إلى 1600x1200.
يتيح ذلك للمستخدمين رؤية العرض الحديث الكامل للصفحة حتى على المتصفحات التي
لا تستطيع عرضه.

### وضع CP-1256 (العربية على Windows القديم)

إذا كان نظامك القديم لا يستطيع عرض النص العربي بترميز Unicode (شائع على
Windows 3.1 مع IE5)، فعّل خانة **CP-1256** قبل تحميل الصفحة. سيُرمّز الجسر
النص العربي بترميز Windows-1256 بدلاً من UTF-8.

### سجل العناوين

يتذكر الجسر العناوين التي تكتبها يدوياً (وليس كل نقرة رابط). يُخزَّن السجل
حسب عنوان IP للعميل، فكل جهاز على الشبكة له قائمته الخاصة. تظهر العناوين
الأخيرة في قائمة منسدلة في الصفحة الرئيسية وشريط التنقل.

### الإعدادات

عدّل الثوابت في أعلى ملف `web_bridge.py`:

| الثابت               | القيمة الافتراضية | الوصف                                  |
|----------------------|-------------------|----------------------------------------|
| `PORT`               | `8888`            | منفذ الاستماع                          |
| `FETCH_TIMEOUT`      | `20`              | مهلة جلب الصفحات بالثواني              |
| `MAX_IMG_W`          | `640`             | أقصى عرض للصور بالبكسل                 |
| `MAX_IMG_H`          | `480`             | أقصى ارتفاع للصور بالبكسل              |
| `MAX_HISTORY`        | `30`              | عدد العناوين المحفوظة لكل مستخدم       |
| `SCREENSHOT_W`       | `800`             | عرض نافذة لقطة الشاشة                  |
| `SCREENSHOT_H`       | `600`             | ارتفاع نافذة لقطة الشاشة               |
| `SCREENSHOT_QUALITY` | `70`              | جودة JPEG للقطات الشاشة (1-100)        |

## إعداد الشبكة

يجب أن يكون جهاز الجسر والحاسوب القديم على نفس الشبكة. اضبط الصفحة الرئيسية
للمتصفح القديم (أو اكتب العنوان مباشرة) على عنوان IP لجهاز الجسر على المنفذ 8888.

مثال -- إذا كان الجسر يعمل على `192.168.1.12`:

```
http://192.168.1.12:8888
```

لا حاجة لتغيير إعدادات المتصفح. يتعامل المتصفح القديم مع الجسر كخادم ويب
عادي، والجسر يجلب المواقع الحقيقية نيابةً عنه.
